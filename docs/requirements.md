# 要件定義

## アプリ概要

Rivalt は、Discord の友達内で行うポーカーの試合結果を記録し、ランキング・個人成績・過去試合を管理するWebアプリです。

## 目的

- 試合結果を簡単に記録する
- 友達内ランキングを自動集計する
- Discordに共有しやすい結果文を生成する
- Excelやスプレッドシートより見やすく管理する

## v1で作る機能

- プレイヤー登録
- 試合結果入力
- ポイント自動計算
- ランキング表示
- 個人成績表示
- 過去試合一覧
- Discord共有文生成
- シーズン管理

## v1で作らない機能

- ポーカーゲーム本体
- チップ推移管理
- 大会機能
- バウンティ大会機能
- Discord Bot連携

## リリース段階（参考）

高レベル要件は v1 として上記に集約する。実装・検証は段階的に行い、現状の機能まとまりは **v0.2**（`docs/roadmap.md`・`docs/test-cases-v0.2.md`）を参照。
