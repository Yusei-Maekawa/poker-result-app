# 設計書

## 全体方針

本アプリは、ポーカーゲーム本体ではなく、ポーカーチェイスなどの既存アプリで遊んだ結果を記録する成績管理アプリとして設計する。

## 利用イメージ

1. ポーカーチェイスなどで友達と対戦する
2. 試合終了後に管理者が順位を入力する
3. ポイントが自動計算される
4. ランキングが更新される
5. Discord共有文をコピーして投稿する

## 権限

### 管理者

- プレイヤー登録
- 試合結果登録
- 試合結果編集
- シーズン管理

### 閲覧者

- ランキング閲覧
- 個人成績閲覧
- 過去試合閲覧

## データ保存方針

データはユーザーごとではなく、リーグごとに保存する。

リーグの中に、プレイヤー、シーズン、試合、試合結果を持たせる。
