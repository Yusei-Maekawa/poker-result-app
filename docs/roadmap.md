## 今後の展望

Poker League Board は、まず v0.1 として通常試合の結果管理を中心に実装します。

v0.1では、ポーカーチェイスなどの既存アプリで遊んだ試合結果を入力し、ランキング・試合一覧・Discord共有文を確認できる状態を目指します。

今後は、実際に友達内で使いながら、必要な機能を段階的に追加していきます。

## 公開予定

Poker League Board は、v0.4 の段階で友達内での本格利用を開始する予定です。

v0.1〜v0.3では、通常試合の結果入力、ランキング表示、シーズン管理などの基本機能を整備します。

v0.4では、プレイヤー詳細画面や直近成績などの個人成績機能を追加し、Discordの友達内で実際に使える状態を目指します。

### v0.1：最小動作版

- プレイヤー登録
- 試合結果入力
- ポイント自動計算
- ランキング表示
- 試合一覧
- Discord共有文生成

### v0.2：使いやすさ改善

- 試合結果の編集・削除
- プレイヤー情報の編集
- 入力ミス防止のバリデーション強化
- スマホ表示の改善
- Discord共有文の見た目調整
- ランキング表示項目の整理

### v0.3：シーズン管理

- Season 1、Season 2 のように成績を分ける機能
- シーズンごとのランキング表示
- 通算ランキングとシーズンランキングの切り替え
- シーズン終了時の最終成績表示

### v0.4：個人成績の強化

- プレイヤー詳細画面
- 直近5試合の成績表示
- 優勝率・入賞率・平均順位の詳細表示
- 連続入賞記録
- プレイヤーごとの傾向メモ

### v0.5：大会・イベント機能

- 通常試合とは別に大会結果を記録できる機能
- バウンティ大会のKO数記録
- 特殊ルール大会の記録
- ポイント倍率の設定
- イベント単位のランキング表示

ただし、v0.1では大会機能は実装せず、通常試合の結果管理に絞ります。

### v0.6：共有・運用機能

- 管理者を複数人設定できる機能
- 閲覧者と管理者の権限分離
- 共有URLの改善
- Firebase Security Rules の強化
- データバックアップ

### v0.7：見た目・盛り上げ機能

- 称号機能
- 月間MVP
- 優勝回数バッジ
- 入賞率ランキング
- グラフ表示
- 試合結果のスクリーンショット添付

### 将来的に検討する機能

- PWA対応
- Discord Bot連携
- OCRによる結果読み取り
- プレイヤー同士の比較機能
- 月間・年間ランキング
- 試合結果のエクスポート機能

### 今後も実装しない機能

以下の機能は、法的リスクや運用トラブルを避けるため、今後も実装しません。

- リアルマネー管理
- 賭け金管理
- 精算機能
- 賞金管理
- 換金可能ポイント
- 金銭の授受を前提とした機能

アプリ内ポイントは、あくまでランキング表示のためのスコアとして扱い、金銭的価値を持たないものとします。
