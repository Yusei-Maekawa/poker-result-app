# 初期モック要件・確認記録

## 1. 目的

Rivalt の初期モックでは、アプリ全体の雰囲気や画面構成を確認することを目的とする。

この段階では、完成版の機能をすべて実装するのではなく、友達に見せたときに以下が伝わる状態を目指す。

- どんなアプリなのか
- どのようにランキングを見るのか
- 試合結果がどのように表示されるのか
- スマホで見やすいか
- 身内ポーカーの成績管理アプリとして使えそうか

## 2. モックの位置づけ

初期モックは、本番利用を前提とした完成版ではなく、画面構成・UI・データ表示の確認用とする。

ただし、将来的にそのまま実装へつなげられるように、React + TypeScript + Tailwind CSS を前提とした構成にする。

## 3. 作成方法

初期モックは、AI によるコード生成を活用して作成した。

- 使用ツール：Claude
- 目的：画面構成とUIの初期案を作成するため
- 対象：Rivalt の v0.1 最小実装
- 保存方式：Firebase Firestore を前提とした実装案

実際に使用したプロンプトは、以下に記録する。

- [使用したAIプロンプト](./used-prompts.md)

## 4. アプリ概要

Rivalt は、Discord の友達内で行うポーカーの試合結果を記録し、ランキング・個人成績・過去試合を管理するWebアプリである。

ポーカーゲーム本体は作成しない。  
実際の対戦は、ポーカーチェイスなどの既存アプリを利用する。

本アプリでは、試合後に順位を入力し、ポイント・優勝回数・平均順位・入賞率などを自動集計して、Discordメンバー内で見やすく共有できるようにする。

## 5. モックで確認したいこと

### アプリ全体

- Rivalt というアプリ名が分かりやすいか
- Discordの友達内ポーカー成績管理アプリであることが伝わるか
- ポーカーゲーム本体ではなく、結果管理アプリであることが伝わるか
- スマホで見ても違和感がないか
- 友達に見せても最低限ちゃんとして見えるか

### UI

- ランキングが見やすいか
- 試合結果が見やすいか
- ボタンの配置が分かりやすいか
- 情報量が多すぎないか
- ダーク系UIとして見やすいか
- ポーカー・リーグ感があるか

### データ表示

- 合計ポイントが分かりやすいか
- 優勝回数が分かりやすいか
- 平均順位が分かりやすいか
- 入賞率が分かりやすいか
- 試合一覧から結果の流れが分かるか

## 6. 作成した画面

### ホーム画面

表示する内容：

- アプリ名
- 現在のシーズン名
- ランキング上位3名
- 直近の試合結果
- ランキング画面へのリンク
- 試合一覧画面へのリンク
- 管理者ログイン時の試合追加導線

目的：

アプリを開いたときに、現在のリーグ状況をすぐに把握できるようにする。

---

### ランキング画面

表示する内容：

- 順位
- プレイヤー名
- 合計ポイント
- 参加回数
- 優勝回数
- 平均順位
- 入賞率
- 最下位回数

目的：

現在の成績順を分かりやすく確認できるようにする。

---

### 試合一覧画面

表示する内容：

- 試合番号
- 日付
- 使用アプリ
- 優勝者
- 参加人数
- 上位3名
- メモ

目的：

過去に行った試合結果を一覧で確認できるようにする。

---

### 試合詳細画面

表示する内容：

- 試合番号
- 日付
- 使用アプリ
- 参加人数
- 順位
- 獲得ポイント
- メモ
- Discord共有文
- コピー操作

目的：

1試合ごとの詳細結果を確認し、Discordに共有しやすくする。

---

### プレイヤー一覧画面

表示する内容：

- プレイヤー名
- アイコン
- 参加回数
- 合計ポイント
- 管理者ログイン時のプレイヤー追加フォーム

目的：

登録されているプレイヤーを確認し、必要に応じて追加できるようにする。

---

### 試合結果入力画面

表示する内容：

- 試合日
- 使用アプリ
- 参加者選択
- 順位入力
- ポイント自動計算プレビュー
- メモ入力
- 保存ボタン

目的：

試合終了後に、管理者が簡単に結果を入力できるようにする。

## 7. 使用するデータ

初期モックでは、以下のデータを扱う。

### プレイヤー

- プレイヤー名
- アイコン
- メモ
- 有効状態
- 作成日時
- 更新日時

### 試合

- 試合番号
- 日付
- 使用アプリ
- メモ
- 作成日時
- 更新日時

### 試合結果

- 試合ID
- プレイヤーID
- 順位
- 獲得ポイント
- 作成日時
